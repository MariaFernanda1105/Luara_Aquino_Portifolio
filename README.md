Luara Aquino — Portfólio Artístico








# 🚀 Sobre o Projeto

Portfólio digital da artista visual Luara Aquino, desenvolvido com HTML, CSS e JavaScript, com foco em experiência visual, narrativa artística e organização de conteúdo.

O projeto traduz sua produção artística e atuação cultural para uma interface web interativa, conectando arte, tecnologia e identidade visual.


# 🧰 Tecnologias Utilizadas
HTML5 (estrutura semântica)
CSS3 (layout, responsividade e estilização)
JavaScript (interatividade e manipulação dinâmica)
SVGs para ilustrações e obras digitais
Design responsivo (mobile-first)
Organização modular de código

# 📌 Organização do Código

O projeto foi estruturado para facilitar manutenção e escalabilidade:

assets/ → elementos visuais e obras em SVG
css/ → estilos globais e específicos do site
js/ → dados e interações dinâmicas
páginas separadas para melhor organização da navegação
